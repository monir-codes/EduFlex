## 📌 Overview
EduFlex is a modern SaaS-based platform built with Next.js for developers to explore learning resources and digital assets.

---

## ⚙️ Tech Stack
- Next.js (App Router)
- Tailwind CSS
- Firebase Auth
- Framer Motion
- Vercel

---

## ✨ Features
- Authentication system
- Modern UI/UX
- SaaS dashboard style
- Animated components
- Fast performance (Next.js optimized)
- Responsive design

## 📦 Dependencies
- next
